# Sense-Coffee

### コンセプト
ロブスタコーヒー種という自国の代表的な特産のテーマを中心し、パッケージモックアップやイラストを制作しました。または、それらのビジュアルを活用したWebサイトも制作し、訪問者に商品の特徴や魅力をわかりやすく紹介しました。

### プログラミング言語、デザインツール
* <ins>**HTML**</ins>:ウェブページの構造や内容を定義するためのマークアップ言語。<br/>
* <ins>**CSS**</ins>: ウェブサイトの詳細の部分をスタイリングのツール<br/>
* <ins>**Illustrator**</ins>: イラスト作成。<br/>
* <ins>**Photoshop**</ins>: モックアップ作成。<br/>
* <ins>**Premiere Pro**</ins>: ビデオ制作・編集。<br/>
